## 第一级：基础核心

> 这是 Python 的地基。没有这些，你连一行有意义的代码都写不出来。所有程序都离不开变量、数据、条件判断和函数，必须最先掌握。

- **Basics**
  - Indentation（缩进）
  - Comments（注释）
  - Variables（变量）
  - Keywords（关键字）

- **Data Types**
  - Numbers: int, float, complex
  - String
  - Boolean
  - None

- **Data Structures**
  - List
  - Tuple
  - Set
  - Dictionary

- **Control Flow**
  - if / elif / else
  - for loop
  - while loop
  - break / continue / pass

- **Functions**
  - def
  - return
  - Arguments & Parameters
  - *args / **kwargs
  - Lambda
  - Scope（global / local）

- **Exception Handling**
  - try / except
  - finally
  - raise

---

## 第二级：进阶工具

> 有了第一级的基础之后，你会开始写更复杂的程序。这时候需要运算符来处理数据之间的关系，需要模块来扩展 Python 的能力，避免重复造轮子。

- **Operators**
  - Arithmetic（算术运算符）
  - Comparison（比较运算符）
  - Logical（逻辑运算符）
  - Assignment（赋值运算符）
  - Identity（身份运算符）
  - Membership（成员运算符）
  - Bitwise（位运算符）

- **Modules & Packages**
  - import
  - from ... import
  - Standard Library

---

## 第三级：高级专题

> 这一级不是"必须会"，而是"会了就能写出专业代码"。OOP 让你能设计大型程序，File Handling 让你能处理真实数据，Advanced 里的内容让你的代码更简洁、更高效、更符合 Python 风格。

- **Object-Oriented Programming (OOP)**
  - class
  - `__init__`
  - Inheritance（继承）
  - Polymorphism（多态）
  - Encapsulation（封装）

- **File Handling**
  - open / read / write / close
  - with 语句

- **Advanced**
  - Comprehensions（列表推导式）
  - Generators（生成器）/ yield
  - Decorators（装饰器）
  - Iterators（迭代器）
