## 一、String 的本质

- 是**数据类型**，不是数据结构
  - `list`, `dict`, `tuple` → 数据结构（用来组织和存储多个数据）
  - `str` → 数据类型（代表一种值的类别）
- 但因为是**可迭代的序列**，所以支持索引、切片、for 循环等操作
  ```python
  s = "hello"
  s[0]      # 'h'  → 支持索引
  s[1:3]    # 'el' → 支持切片
  len(s)    # 5    → 支持长度
  for c in s        # 可以迭代
  ```
- 空字符串可以用三种方式创建
  ```python
  s = ""      # 方式一：双引号
  s = ''      # 方式二：单引号
  s = str()   # 方式三：调用构造函数
  ```

## 二、可变 vs 不可变

- `str` **不可变**：创建后不能修改，只能生成新字符串
  ```python
  s = "hello"
  s[0] = "H"        # ❌ 报错！TypeError
  s = "H" + s[1:]   # ✅ 只能生成新的 "Hello"
  ```
- `list` **可变**：创建后可以直接修改元素
  ```python
  nums = [1, 2, 3]
  nums[0] = 99      # ✅ 没问题
  print(nums)       # [99, 2, 3]
  ```
- 比喻
  - `str` 不可变 → 印刷好的书，想改只能重新印一本
  - `list` 可变 → 笔记本，随时可以涂改、添加、删除
- 注意：写 `s = s + "!"` 时，Python 并不是在原字符串上修改，而是**新建了一整个字符串**，再把变量 `s` 指向它

## 三、迭代（iteration）

- 意思是**一个一个地取出来遍历**，即 for 循环
  ```python
  s = "hello"
  for c in s:
      print(c)
  # 输出：h e l l o
  
  nums = [10, 20, 30]
  for n in nums:
      print(n)
  # 输出：10 20 30
  ```
- 词根拆解：`iter`（拉丁语，意为"再次"）+ `-ation`（名词后缀，表示动作/过程）
  - 字面意思：**一次又一次地做某件事的过程**
- 相关词
  - `iterate` → 动词，反复做
  - `iterable` → 形容词，可迭代的（str、list、tuple 都是）
  - `iterator` → 名词，迭代器
  - `iterative` → 形容词，反复的
  - `reiterate` → 再次强调（re- 又加了一个"再次"，语气更强）
- 常见报错
  ```
  TypeError: 'int' object is not iterable
  ```
  意思是：整数不能被迭代，不能用 for 循环遍历一个数字

## 四、为什么 input() 返回的是 string

- 因为 input() 读取的是"键盘输入"，键盘输入的本质是**字符**，不是数字或其他类型
  - 当你按下 `1`、`2`、`3` 这几个键，计算机收到的是字符 `"1"`、`"2"`、`"3"`，而不是数字 `123`
- 比喻：给朋友发短信写了 `123`，朋友收到的是**文字"123"**，而不是数学上的数字 123，要变成数字需要自己转换
- input() 只负责**原样收下你打的字**，不做任何判断和转换

- 转换类型需要手动告诉 Python
  ```python
  x = input("请输入一个数字：")   # x 是 str，比如 "42"
  x = int(x)                      # 转成 int → 42
  x = float(x)                    # 转成 float → 42.0
  ```
  或者直接套在一起写：
  ```python
  x = int(input("请输入一个数字："))
  ```

- Python 不自动判断类型，因为**自动判断会产生歧义**
  - 用户输入 `123` → 是 int？还是 float？还是就想要字符串？
  - 用户输入 `3.14` → 是 float？还是字符串？
  - 用户输入 `hello` → 显然只能是字符串
- Python 的设计哲学：**明确优于隐式（Explicit is better than implicit）**，所以它不猜，让你自己决定
