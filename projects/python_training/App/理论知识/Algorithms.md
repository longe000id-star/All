## 第一级：基础算法

> 这是算法思维的起点。学会这些，你才能理解"怎么让程序更快、更省力"。所有复杂算法都建立在这些基础概念之上，必须最先掌握。

- **Complexity（复杂度）**
  - Time Complexity（时间复杂度）
  - Space Complexity（空间复杂度）
  - Big O Notation（大O表示法）

- **Sorting（排序）**
  - Bubble Sort（冒泡排序）
  - Selection Sort（选择排序）
  - Insertion Sort（插入排序）

- **Searching（搜索）**
  - Linear Search（线性搜索）
  - Binary Search（二分搜索）

- **Recursion（递归）**
  - Base Case（基础情况）
  - Recursive Case（递归情况）
  - Call Stack（调用栈）

---

## 第二级：进阶算法

> 有了基础之后，你会开始面对更复杂的问题。这一级的算法和数据结构是大多数实际项目和技术面试的核心，学完之后能解决绝大多数编程问题。

- **Advanced Sorting（进阶排序）**
  - Merge Sort（归并排序）
  - Quick Sort（快速排序）
  - Heap Sort（堆排序）

- **Data Structure Algorithms（数据结构算法）**
  - Stack（栈）
  - Queue（队列）
  - Linked List（链表）
  - Hash Table（哈希表）

- **Tree Algorithms（树算法）**
  - Binary Tree（二叉树）
  - Binary Search Tree（二叉搜索树）
  - Tree Traversal（树的遍历）：DFS / BFS

- **Graph Algorithms（图算法）**
  - DFS（深度优先搜索）
  - BFS（广度优先搜索）
  - Dijkstra（最短路径）

---

## 第三级：高级算法

> 这一级是算法学习的顶端。不是每个人都需要全部掌握，但懂得越多，解决复杂问题的能力就越强。适合准备大厂面试或深入研究计算机科学的人。

- **Dynamic Programming（动态规划）**
  - Memoization（记忆化）
  - Tabulation（制表法）
  - 经典问题：Fibonacci、Knapsack（背包问题）

- **Greedy Algorithms（贪心算法）**
  - 经典问题：Coin Change、Activity Selection

- **Backtracking（回溯）**
  - 经典问题：N-Queens、Sudoku

- **Advanced Graph Algorithms（高级图算法）**
  - Bellman-Ford
  - Floyd-Warshall
  - Minimum Spanning Tree: Kruskal / Prim

- **Bit Manipulation（位运算算法）**
  - AND / OR / XOR / Shift
