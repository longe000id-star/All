# Quiz 评分配置

## 评分原则

### 1. 文件修改规则
- **严禁修改原测试文件** (`quiz*.md`)
- 只能在评分报告中提供参考答案
- 评分报告文件名格式：`quiz*_score_report.md`

### 2. 评分标准


### 3. 评分要求

#### 错误分析
- 详细列出代码中的所有错误
- 按类型分类（语法错误、逻辑错误、功能缺失等）
- 提供具体的错误位置和修复建议

#### 参考答案
- 在评分报告中提供完整的参考实现
- 代码必须能正常运行
- 包含所有要求的功能

#### 改进建议
- 根据错误类型提供针对性的学习建议
- 从基础语法到框架使用，循序渐进

### 4. 评分等级
- **90-100分**: 优秀
- **80-89分**: 良好
- **70-79分**: 中等
- **60-69分**: 及格
- **60分以下**: 不及格
